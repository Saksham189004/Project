# Project
# 🎮 Esports Tournament Viewer

## 📌 Project Overview

Esports Tournament Viewer is a web application that displays real-time esports match data using the PandaScore API.
Users can explore upcoming matches, search for teams, filter by game, and sort matches by date.

This project demonstrates the use of JavaScript, API integration, and dynamic UI development.

---

## 🚀 Features

* 🔍 **Search Matches** – Search matches by team or match name
* 🎮 **Filter by Game** – Filter matches (Valorant, CSGO, etc.)
* ⬆️ **Sort by Date** – Arrange matches based on schedule
* ❤️ **Favorites** – Save matches using Local Storage
* ⏳ **Loading Indicator** – Shows loading state during API fetch
* 📱 **Responsive Design** – Works on mobile, tablet, and desktop

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 🔗 API Used

* PandaScore API
  Provides real-time esports data including matches, teams, and tournaments.

---

## 📦 How to Run the Project

1. Clone the repository
2. Open the project folder
3. Add your API key in `script.js`
4. Open `index.html` in your browser

---

## ⚠️ Important Note

Make sure to replace the API key in the code with your own PandaScore API key.
